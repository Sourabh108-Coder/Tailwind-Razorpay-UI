# 💳 Razorpay Landing Page UI (Tailwind CSS)

A modern, fully responsive landing page UI inspired by Razorpay, built using Tailwind CSS.  
This project focuses on clean design, reusable components, and a fintech-style layout with smooth responsiveness.

---

## 📸 Project Preview



---

## 🛠️ Tech Stack

- HTML5  
- Tailwind CSS (Installed via npm)  
- Responsive Web Design  
- Flexbox & Grid Layout  

---

## 🎯 Features

- Modern Razorpay-inspired UI  
- Fully responsive design (mobile, tablet, desktop)  
- Clean and reusable components  
- Smooth hover animations  
- SaaS-style landing page layout  
- Fast and lightweight UI  
- Well-structured sections  
- Beginner-friendly Tailwind implementation
